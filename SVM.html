Summary

For the Python part, I think we get a great SVM model with a linear kernel(C=10), the accuracy is around 70%. When we changed the kernel, the confusion matrix did not change very much. However, when we add the C value, the SVM model gives us bad prediction results with accuracy less than a half. I think we should not make the C value to big to create my model in this case.

For my R part, as I mentioned before, when the cost changes from 5% to 10%, the accuracy has a great improvement. However, the model does not work if cost is equal to 30%. This means making the cost greater can have a better model but we should not set it too much. We need to find the best interval of the cost value to get the greatest model. In general, our model is good with an accuracy of 57%.
